1. 사전 준비 및 설치 (Prerequisites)
	이 프로젝트를 실행하려면 컴퓨터에 Python이 설치되어 있어야 합니다.
	설치 후, 터미널(명령 프롬프트)을 열고 아래 명령어를 복사하여 필요한 라이브러리를 설치해 주세요.
	pip install flask flask-cors opencv-python numpy

2. 필수 파일 구성
	실행 전, 하나의 폴더 안에 아래 파일들이 모두 모여 있는지 확인하세요.
	- app.py: 이미지 윤곽선을 추출해 전달하는 파이썬 백엔드 서버 파일
	- 26.06.21 ai art.html: 메인 애니메이션이 구동되는 프론트엔드 캔버스 파일
	- input1.jpg, input2.jpg, input3.jpg: 형태를 추출할 원본 타겟 이미지 파일 (확장자가 코드와 일치해야 합니다)

3. 실행 방법 (How to Run)
	1단계: 파이썬 서버 켜기
		터미널을 열고 현재 폴더 경로로 이동한 뒤, 아래 명령어를 입력하여 서버를 실행합니다.
		python app.py

	2단계: 캔버스 열기
		웹 브라우저(Chrome 권장)에서 26.06.21 ai art.html 파일을 더블클릭하여 엽니다.

	3단계: 시뮬레이션 시작
		화면 정중앙의 [START] 버튼을 클릭하여 작품을 감상합니다.

4. 종료 방법
	시뮬레이션 창(브라우저)을 닫습니다.
	켜두었던 터미널 창을 클릭하고 키보드의 Ctrl + C를 누르거나, 창을 닫아 서버를 종료합니다.
